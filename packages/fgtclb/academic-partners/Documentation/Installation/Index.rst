============
Installation
============

This extension can be installed using the TYPO3 `extension manager
<https://extensions.typo3.org/extension/academic_partners>`__ or by  `composer
<https://packagist.org/packages/fgtclb/academic-partners>`__.

..  code-block:: shell

    composer install fgtclb/academic-partners
