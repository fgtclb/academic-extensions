=====================
Academic Persons Edit
=====================

:Extension key:

----

This extension extends the `Academic Persons
<https://extensions.typo3.org/extension/academic_persons>`__ extension by
connecting person profiles to frontend users. It provides plugins to edit the
assigned profiles from the TYPO3 frontend.
